<h1>💡 프로젝트 소개 및 목표</h1>
  <p>팀 구성원 모두 온라인 쇼핑몰 사이트의 동작 원리에 대한 궁금증이 있었기에 이를 탐구해보고 여태 학원에서 학습한 내용들을 복습하는 취지로 게임 관련 굿즈를 판매하는 온라인 쇼핑몰을 개발 했습니다.</p>
  <p>개발이 필요한 페이지들을 팀원끼리 분담했고, 각자 구현 해보고 싶은 기능들을 나열하여 해당 페이지들에 적용했습니다.</p>
